Flower Haven - Site Plan

Site Name

Flower Haven

This name represents a welcoming and comprehensive resource for flower enthusiasts. It aims to be a haven for anyone interested in learning about and appreciating the beauty of flowers.

Site Purpose

The site provides detailed information on various flowers, allowing users to explore flower characteristics, navigate through different categories, mark favorites, and engage with interactive features like quizzes.

Scenarios

  • Which flowers are best suited for a sunny garden?
  • How can I identify a flower I found in my backyard?

Color Schema

Primary Color: (#D61355) - Used for headings.

Secondary Color: (#F94A29) - Used for accents and highlights.

Text Color: (#FCE22A) - Used for body text.

Background Color: (#30E3DF) - Used for page background.

Typography

Headings: Montserrat, sans-serif - Clear and readable, used for all headings (h1, h2, h3).

Body Text: Montserrat, sans-serif - Consistent with headings, used for all body text and paragraphs.
